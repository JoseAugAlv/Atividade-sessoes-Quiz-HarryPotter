# Quiz de Harry Potter - Seleção de Casas de Hogwarts

Este projeto é um sistema de quiz desenvolvido em PHP com foco em lógica de sessão, interação com o usuário e manipulação de dados no lado do servidor.

O objetivo do sistema é simular a famosa seleção de casas de Hogwarts, onde o usuário responde a perguntas e recebe uma casa baseada em sua pontuação.

## Funcionalidades

- Cadastro do nome do usuário via formulário
- Armazenamento de dados utilizando sessões em PHP
- Sistema de pontuação para as casas:
  - Grifinória
  - Sonserina
  - Corvinal
  - Lufa-Lufa
- Validação de formulário
- Feedback visual com SweetAlert2
- Contagem regressiva antes do início do quiz
- Redirecionamento automático para a primeira pergunta

## Tecnologias utilizadas

- PHP (sessões e lógica do backend)
- HTML5
- CSS3
- JavaScript
- SweetAlert2

## Estrutura do projeto

- Página inicial com introdução ao quiz
- Formulário de entrada do nome do usuário
- Sistema de validação de dados
- Tela de apresentação das casas de Hogwarts
- Início automático do quiz após confirmação

## Objetivo educacional

Este projeto foi desenvolvido como atividade escolar para praticar:

- Lógica de programação em PHP
- Uso de sessões
- Manipulação de formulários
- Integração entre frontend e backend
- Uso de bibliotecas externas em JavaScript
